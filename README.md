# Mario-like Platformer Template (Non-copyrighted assets)

## Quick start
1. Copy `.env.example` to `.env` and set BOT_TOKEN and WEBAPP_URL (if using Telegram WebApp button)
2. npm install
3. npm start
4. Open http://localhost:3000 to test the game in browser
5. To test inside Telegram (WebApp), use ngrok: `ngrok http 3000` and set WEBAPP_URL to the ngrok HTTPS URL, then update BotFather or use bot to send web_app button.

## Notes
- Assets included are basic placeholders generated programmatically. They are NOT Nintendo assets and are free to use.
- To use your own graphics, replace files in public/assets as described in public/README_ASSETS.txt
